# GeoBase: 几何算法基础库
个人学习项目，C++ 工业级三维几何算法基础工具库，面向 CAD / 工业软件 / 几何算法岗开发。
已实现功能
Vector3（三维向量）

基础构造、加减乘除、标量运算
点积、叉积、模长计算、单位化
鲁棒性运算支持
全量单元测试覆盖

Matrix3（3×3 变换矩阵）

单位矩阵、旋转矩阵（X/Y/Z 轴）
矩阵乘法、矩阵转置
与 Matrix4 双向转换
全量单元测试覆盖

Matrix4（4×4 变换矩阵）

平移、旋转、缩放基础变换
矩阵乘法、转置、浮点近似相等判断
点 / 方向向量变换（区分点与向量）
仿射矩阵求逆（inverseAffine）
观察矩阵 LookAt
全量单元测试 + 边界 / 退化场景测试

核心特性

纯 C++ 实现，无第三方依赖
鲁棒浮点精度处理
工业几何算法常用线性代数底层
完整单元测试（GTest）
适配三维建模、网格处理、坐标变换
