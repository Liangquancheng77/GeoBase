# GeoBase: 几何算法基础库

个人学习项目，目标是为几何算法 / 工业软件开发一套可复用的 C++ 基础工具库。

## 已实现功能

### Vector3（三维向量）
- 构造函数、加减乘除运算符重载
- 点积（dot）、叉积（cross）
- 模长（length）、归一化（normalize）
- 单元测试覆盖（GTest）

## 学习路线

- 基础线性代数（向量、矩阵、变换）
- 计算几何常用算法（线段求交、凸包、Delaunay 三角剖分）
- 网格数据结构与处理（半边结构、简化、修复）
