# GeoBase：几何算法基础库
个人学习项目，基于 C++ 实现的工业级三维几何算法工具库。
面向 CAD、工业软件、三维仿真、几何算法求职方向打造。

## 已实现功能

### Vector3 三维向量
- 基础构造、四则运算、标量运算
- 点积、叉积、模长、归一化
- 浮点鲁棒性处理
- 全套 GTest 单元测试覆盖

### Matrix3 3×3 旋转矩阵
- 单位矩阵、X/Y/Z 轴旋转矩阵
- 矩阵乘法、转置、正交判断
- 工业标准 Y-X-Z 欧拉角双向转换
- 与 Matrix4 互相转换
- 全量单元测试 + 边界场景测试

### Matrix4 4×4 仿射矩阵
- 平移、旋转、缩放基础变换
- 矩阵乘法、转置、浮点近似相等
- 点与方向向量变换区分
- 仿射矩阵求逆、LookAt 观察矩阵
- 完整单元测试与退化场景覆盖

### Quaternion 四元数
- 欧拉角 / 矩阵 / 四元数 两两互转
- 四元数乘法、共轭、求逆、归一化
- 矩阵转四元数采用最大分量分支，保证数值稳定
- 非正交矩阵合法性校验
- 零四元数异常保护
- 全套 GTest 测试
